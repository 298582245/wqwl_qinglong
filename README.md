# wqwl_qinglong

一些辣鸡的青龙脚本，QQ 裙：960690899

## 💡 食用指南

- 1. 下载 [wqwl_require.js](https://github.com/298582245/wqwl_qinglong/blob/main/wqwl_require.js)
- 2. 下载需要的脚本到相同的目录下
- 3. 配置脚本需要的环境变量
- 4. 运行脚本
- ps. 文件名以 wqwl_new 为前缀的，需要的 wqwl_require 必须包含 WQWLBase 类、WQWLBaseTask 类（2025-11-18 之后的版本）
